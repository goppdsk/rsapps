# Yew Todo frontend application

## Dependencies

- [Yew](https://yew.rs/docs/en/)
- [trunk](https://github.com/thedodd/trunk)
- [strum](https://docs.rs/strum/0.20.0/strum/index.html)

```bash
cargo install trunk
```

## Build

```bash
trunk build --release
```

## Serve on localhost

```bash
trunk serve --release
```
